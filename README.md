# Full Stack Dev Challenge

This is a full stack web app development challenge. In this challenge I will try to implement the same app with different technologies.

# Technologies

- ASP.net Web Form
- ASP.net MVC 5
- ASP.net Core
- ASP.net Web API
- SQL Server
- MERN Stack (MongoDB, Express, React, Node)
- PHP
- Drupal CMS
